1. The Converter Problem breaks down a large number of seconds into minutes, hours and days
<br />
2. It accepts one positive number as input, which should represent a number of seconds.  
<br />
3. It outputs the time in this format Days:Hours:Minutes:Seconds, where seconds always includes at least one decimal place.
<br/>
4. If input is not positive it will output, "Error: Input must be positive."
<br/>
5. Other invalid inputs will output, "Error: Bad Input".
